# Changelog — HKUST-1 MOF Structure Editor

Entries are added daily by the implementation routine. Each entry covers user-visible changes only.
Internal refactoring, test additions, and dependency updates are omitted.

---

## 2026-04-21

### New
- Polyhedra rendering in WebGL renderer (`?renderer=three`): coloured planes and rings now appear as translucent face meshes with solid edges, matching the default Canvas 2D view.

---

## 2026-04-20

### New
- **Three.js bond rendering** (`?renderer=three`): bonds between atoms are now drawn as 3D cylinders. Dashed bonds appear thinner and darker than solid bonds.
