## Architectural notes
- `getAllDrawGroups()` already returns sphere (cavity) groups mixed with polyhedron groups. Cavities have `isSphere:true` — skip them for now (they'll get spherical mesh treatment in a later A1.x slice; today is polyhedra only).
- Faces are arbitrary planar polygons (3+ vertices). Use simple fan triangulation from `faces[i][0]` to remaining vertices — `convexHull3DFaces` already returns ordered vertices, so fan tri is safe.
- Use `MeshBasicMaterial` (not lit) with `opacity ~0.25`, `transparent:true`, `side: DoubleSide`, `depthWrite:false` to match Canvas 2D's translucent face look.
- Edges: `LineBasicMaterial` + `BufferGeometry.setFromPoints([v1,v2,...])` then `LineSegments` — one geometry per group is fine, no need for per-edge instancing yet.
- Mesh rebuild every frame remains intentional for Phase A1; caching/dirty-flag belongs to A2.
- Vertex (`type:'vtx'`) highlights from Canvas 2D draw list are deferred — they're tied to selection/hover state which lives in A2 picking.
