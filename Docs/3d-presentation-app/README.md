## One-liner

> PowerPoint, but the slides are positions on an infinite 3D plane — and every object on them is live and interactive.

---

## Key Strategic Bets

1. **Multi-origin X-Y navigation** — arrow keys navigate between spatial anchors like slides
2. **Scene annotation layer** — text and image overlays anchored to 3D space
3. **Scene transition export** — record camera glide between origins as WebM/GIF

## Non-Goals

- Z-axis navigation
- Real-time collaboration
- Cloud accounts / storage
- Audio narration
- PPTX import (Phase 4+)
